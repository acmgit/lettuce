# Lettuce


This Mod adds Lettuce to Minetest.<br>
It supports farming and farming_redo from TenPlus.

![Screenshot 1](textures/lettuce_screenshot.jpg)

## Install

Move your Download to the Mods-Folder.

## Depends

default<br>
vessels<br>
farming<br>

### optional
farming_redo<br>
vegan_sunflower
